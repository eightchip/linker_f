# 予定管理機能の提案

## 概要
タスクに複数の予定日時（会議日時など）を登録し、予定表として一覧表示できる機能を実装します。

## 設計方針
- **スキーマ変更なし**: 既存のTaskItemスキーマは変更せず、別のHiveボックスで予定データを管理
- **データモデル**: `ScheduleItem`クラスを作成し、`taskSchedules`というHiveボックスで管理
- **予定の種類**: タスクに関連する予定（会議、打ち合わせなど）を複数登録可能（制限なし）

## データモデル

```dart
class ScheduleItem {
  String id;                    // 予定ID
  String taskId;               // 関連するタスクID
  String title;                // 予定タイトル（タスク名から自動生成、編集可能）
  DateTime startDateTime;      // 開始日時
  DateTime? endDateTime;        // 終了日時（オプション）
  String? location;            // 場所（オプション）
  String? notes;               // メモ（オプション）
  DateTime createdAt;          // 作成日時
  DateTime? updatedAt;         // 更新日時
}
```

## 機能要件

### 1. 予定の登録・編集・削除
- タスク編集モーダルに「予定」セクションを追加
- 予定の追加・編集・削除が可能
- 日時選択、終了日時（オプション）、場所、メモを入力可能

### 2. 予定表の表示
- 新しい「予定表」画面を作成
- 日付順に予定を一覧表示
- 各予定にタスク名を表示
- 予定をクリックすると関連するタスクの編集モーダルを開く
- タスク名をクリックしてもタスクにアクセス可能

### 3. 予定表へのアクセス
- タスク管理画面から「予定表」メニューを追加
- ショートカットキー（例: Ctrl+Shift+C）で予定表を開く

## 実装ファイル

1. `lib/models/schedule_item.dart` - 予定データモデル
2. `lib/viewmodels/schedule_viewmodel.dart` - 予定のCRUD操作
3. `lib/views/schedule_calendar_screen.dart` - 予定表画面
4. `lib/views/task_dialog.dart` - 予定セクションの追加

## UI設計

### タスク編集モーダルの予定セクション
- 折りたたみ可能なセクション
- 予定一覧（日時、タイトル、場所）
- 「予定を追加」ボタン
- 各予定に編集・削除ボタン

### 予定表画面
- カレンダー形式またはリスト形式
- 日付ごとにグループ化
- 予定アイテムにタスク名、時間、場所を表示
- クリックでタスク編集モーダルを開く
