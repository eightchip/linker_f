# Link Navigator 取扱説明書

## 基本情報
- **アプリ名**: Link Navigator
- **概要**: ファイル・フォルダ・URLをグループで整理し、ドラッグ＆ドロップで直感的に管理できるデスクトップアプリケーション
- **対応OS**: Windows 11

## 主要機能一覧
1. グループ管理機能
2. リンク管理機能（ファイル・フォルダ・URL）
3. ドラッグ＆ドロップ操作
4. 検索機能
5. メモ機能
6. テーマ切り替え（ライト/ダークモード）
7. アクセントカラー変更
8. データのエクスポート・インポート
9. メモ一括編集
10. タスク管理機能
11. サブタスク管理機能
12. 通知機能（音声付き）
13. キーボードショートカット
14. Googleカレンダー連携機能（※利用にはユーザー側での設定が必要）
15. Outlook予定取り込み機能
16. 予定表（スケジュール一覧）機能
17. Gmail連携機能（メーラー起動）
18. ヘルプセンター（Markdown表示・HTML出力）

### リンク管理画面
- グループとリンクを管理するメインビューです。
- 画面右上の**3点ドットメニュー**から追加・設定・ヘルプ・ショートカットなどの操作を行います。
- `Ctrl+N` でグループ追加、`Ctrl+F` で検索バー表示など主要操作にショートカットが設定されています。

#### 3点ドットメニュー（主な項目）
- `[リンク管理 3点メニューのスクリーンショットを見る](screenshot:link-menu)`
- **グループを追加**: 新しいグループを作成
- **検索**: 検索バーの表示/非表示を切り替え
- **タスク管理**: タスク管理画面へ遷移
- **メモ一括編集**: メモが登録されているリンクをまとめて確認・編集
- **ショートカットキー**: `?` アイコンと同内容のショートカット一覧ダイアログを開く
- **設定**: テーマ・カラープリセット・データ管理など各種設定を行う
- **ヘルプセンター**: 本マニュアルを表示

### タスク管理画面
- リンク管理画面から遷移するタスク集中操作エリアです。
- `F1` もしくは AppBar の `?` アイコンでショートカット一覧を表示できます。

#### タスクリストビュー（既定表示）
- タスクの詳細を縦型リストで表示。
- 画面右上の3点メニューからグリッドビューやスケジュール一覧に切り替え。

#### タスクグリッドビュー
- `[タスクグリッドビューのスクリーンショットを見る](screenshot:task-grid)`
- タスクをカード形式で俯瞰するビュー。進捗や分類を視覚的に確認できます。

#### スケジュール一覧
- `[スケジュール一覧のスクリーンショットを見る](screenshot:schedule-list)`
- タスクと紐付いた予定を日付順に確認。画面右上のメニューでリスト/週次/月次表示を切り替え可能です。

## 基本操作手順

### グループの作成
1. リンク管理画面右上の3点ドットメニューで「グループを追加」を選択
2. グループ名を入力
3. 色やアイコンを必要に応じて設定
4. 「作成」をクリック

### リンクの追加

#### 方法1: 手動追加
1. グループの「＋」アイコンをクリック
2. リンクタイプを選択（ファイル/フォルダ/URL）
3. ラベル（表示名）を入力
4. パス/URLを入力
5. 「追加」をクリック

#### 方法2: ドラッグ＆ドロップ
1. エクスプローラーからファイル/フォルダをドラッグ
2. 対象グループにドロップ
3. ブラウザからHTML形式で書き出したブックマークファイル（*.html）も同様にドラッグ＆ドロップで登録可能
   - 書き出し手順はブラウザごとに異なるため、必要に応じて「[ブックマーク エクスポート方法を調べる](https://www.google.com/search?q=%E3%83%96%E3%83%A9%E3%82%A6%E3%82%B6+%E3%83%96%E3%83%83%E3%82%AF%E3%83%9E%E3%83%BC%E3%82%AF+%E3%82%A8%E3%82%AF%E3%82%B9%E3%83%9D%E3%83%BC%E3%83%88+%E6%96%B9%E6%B3%95)」をクリックして最新情報を確認してください

### リンクの操作

#### 起動
- リンクをクリック

#### 編集
- 「編集」アイコンをクリック

#### 削除
- 「削除」アイコンをクリック

#### メモ追加
- 「メモ」アイコンをクリック

### グループの操作

#### 並び替え
- グループをドラッグ＆ドロップで移動

#### 編集
- グループの「編集」アイコンをクリック

#### 削除
- グループの「削除」アイコンをクリック

### リンクの移動
- リンクをドラッグして他のグループにドロップ

## 検索機能
1. `Ctrl+F` または 3点ドットメニューで「検索」を選択し検索バーを表示
2. キーワードを入力（ファイル名・フォルダ名・URL・説明文・タグが対象）
3. 入力と同時に結果がフィルタリングされる
   - ハイライトされるのは「タイトル」と「説明」部分のみです（タグや依頼先などは結果に含まれますが強調表示されません）

## メモ機能

### 個別メモ
1. リンクの「メモ」アイコンをクリック
2. メモ内容を入力
3. 「保存」をクリック

### 一括メモ編集
1. 3点ドットメニューで「メモ一括編集」を選択
2. メモが登録されているリンクが一覧表示される
3. 各リンクのメモを編集
4. 「まとめて保存」をクリック

## テーマ設定

### ライト/ダークモード切り替え
1. リンク管理画面の3点ドットメニューから「設定」を開く
2. 「テーマ」セクションのトグルでライト/ダークモードを切り替え

### アクセントカラー変更
1. リンク管理画面の3点ドットメニューから「設定」を開く
2. 「アクセントカラー」のカラーパレットから色を選択
3. 「選択」をクリック

### カラープリセット
- 設定画面の「カラープリセット」からおすすめ配色（9種類）をワンタップで適用
- プリセットには濃淡・コントラスト・ダークモード推奨設定が含まれます

## データ管理

### エクスポート
1. リンク管理画面の3点ドットメニューから「設定」を開く
2. 「データ管理」セクションの「エクスポート」をクリック
3. 保存場所とファイル名を指定し「保存」をクリック

### インポート
1. リンク管理画面の3点ドットメニューから「設定」を開く
2. 「データ管理」セクションの「インポート」をクリック
3. エクスポート済みファイルを選択し「開く」をクリック
4. 既存データとの統合方法を指定して実行

### バックアップ機能
- **自動バックアップ**: アプリ起動時にチェックし、設定された間隔で実行
- **手動バックアップ**: 設定画面の「手動バックアップ実行」ボタンで実行可能
- **バックアップフォルダ**: 設定画面の「バックアップフォルダを開く」ボタンでフォルダを開く
- **保存場所**: %APPDATA%/linker_f/backups/
- **ファイル形式**: JSON（リンク、タスク、設定を含む）
- **最大保存数**: 10個（古いものは自動削除）

## タスク管理機能

### タスクの作成
1. リンク管理画面の3点ドットメニューから「タスク管理」を開く
2. タスク管理画面右上の「新しいタスク」ボタンをクリック（または `Ctrl+N`）
3. タスク情報を入力（タイトル、説明、期限、優先度など）
4. 「作成」をクリック

### タスクの操作
- **編集**: タスクをクリックして編集
- **削除**: タスクの3点メニューから削除（完了済タスクも削除可能）
- **ステータス変更**: 未着手 → 進行中 → 完了
- **優先度設定**: 低、中、高、緊急（色アイコン＋文字で表示）
- **ピン留め**: タスクを上部に固定表示
- **一括操作**: 複数タスクを選択して一括削除・ステータス変更・優先度変更
- **グループ化**: 期限日、タグ、リンク、ステータス、優先度でグループ化表示
- **フィルター**: ステータス、優先度、タグ、期限日色でフィルタリング
- **検索**: タスクタイトル、説明、タグで検索（ハイライト表示）

### タスクの並び替え
- **期限順**: 期限の昇順で表示（期限なしは最後）
- **優先度順**: 緊急度の高い順で表示
- **タイトル順**: アルファベット順で表示
- **作成日順**: 作成日の新しい順で表示
- **ステータス順**: 未着手→進行中→完了の順で表示
- **手動調整**: 各カード右端のドラッグハンドル（⋮⋮）を長押しして上下にドラッグ → 任意の順序に変更（クリックは編集）
- **複数条件**: 第1順位、第2順位、第3順位まで設定可能

### サブタスク管理機能
- **サブタスクの追加**: タスクのサブタスクアイコンをクリック
- **サブタスクの編集**: サブタスクダイアログで詳細編集
- **進捗表示**: 完了数/総数のバッジ表示
- **見積時間**: 各サブタスクの見積時間設定
- **メモ機能**: サブタスクごとのメモ追加

### リマインダー機能
- タスクに期限を設定すると自動的にリマインダーが設定される
- 通知音付きのデスクトップ通知でリマインダーを表示
- 繰り返しリマインダー設定可能（日次、週次、月次など）

### タスクとリンクの連携
- タスクにリンクを関連付け可能
- リンクの並び替え（ドラッグ&ドロップ）
- リンクのメモ内容をタスクカードでホバー表示
- リンクのfavicon表示（URLはフォールバックfavicon）
　※フォールバックfaviconは実際に表示したいアイコンのURLを入力　

## Googleカレンダー連携機能

### 連携設定
- **重要**: 初期状態では無効です。Google カレンダー連携を利用する場合は、ユーザー自身で Google Cloud Console にて OAuth クライアントを作成し、設定画面に認証情報を登録する必要があります。
1. 設定画面の「Google Calendar」セクションを開く
2. 取得したクライアントID/シークレットを入力し、認証を実行
3. 自動同期間隔を設定（分単位）

### 同期機能
- **完全同期**: アプリとGoogleカレンダーの双方向同期
- **祝日除外**: 祝日イベントは自動的に除外
- **重複防止**: 同じタスクの重複作成を防止
- **起動時自動同期**: アプリ起動時の自動同期

### 同期内容
- **アプリ→Googleカレンダー**: タスクをカレンダーイベントとして作成
- **Googleカレンダー→アプリ**: カレンダーイベントをタスクとして取得
- **祝日除外**: 祝日関連のイベントは同期対象外
- **予定の同期**: 予定表の予定もGoogleカレンダーに自動同期

## Outlook予定取り込み機能

### 機能概要
- Outlookカレンダーから予定を自動取得
- タスクと予定を関連付け
- 予定の変更を自動検出して更新

### 予定の取り込み方法

#### 方法1: タスク編集モーダルから
1. タスク編集モーダルを開く
2. 「Outlookから予定を取り込む」ボタンをクリック
3. 期間を選択（デフォルト：明日から1か月間）
4. 「予定を取得」ボタンをクリック
5. 取り込む予定を選択
6. 既存タスクに割り当て、または新規タスクを作成

#### 方法2: 予定表から一括取り込み
1. 予定表画面の「Outlookから予定を取り込む」ボタンをクリック
2. 期間を選択（開始日・終了日を個別設定可能）
3. 「予定を取得」ボタンをクリック
4. 非同期処理で予定を取得・照合（メッセージ・アニメーション表示）
5. 変更あり/未取込の予定のみ表示
6. 検索・ソート機能で予定を絞り込み
7. チェックボックスで予定を選択
8. 「タスクに割り当て」または「新規タスク作成」をクリック

### 照合機能
- **既存予定の検出**: `outlookEntryId`で取込済み予定を識別
- **変更検出**: 時間・場所の変更を自動検出
- **完了タスク除外**: 完了済みタスクに関連する予定は除外
- **重複防止**: 既に取り込まれている予定は表示しない（変更がある場合のみ表示）

### 表示機能
- **場所未指定の予定**: 場所が指定されていない予定も表示対象
- **ソート**: タイトル昇順→日時昇順、または日時昇順
- **検索**: 予定タイトル、場所で検索

## 予定表（スケジュール一覧）機能

### 機能概要
- タスクに関連付けられた予定を一覧表示
- 日付別に整理された表示
- 予定の編集・削除・コピー機能

### 予定表の表示
1. ホーム画面のサイドバーから「予定表」を開く
2. 日付別に予定が表示される
3. デフォルトで今日以降の予定のみ表示
4. 「過去を表示」チェックボックスで過去の予定も表示可能
5. 右上の「表示切り替え」メニューから以下を選択可能
   - **リスト表示**: 日付ごとに予定カードを表示（既定）
   - **週次表示**: 1週間単位で日時とタイトルをコンパクトに一覧表示
   - **月次表示**: 月単位で各日の予定を一覧表示（日時＋タイトルのみ）

### 予定の操作
- **追加**: FABボタンで予定を追加
- **編集**: 予定を長押ししてメニューから編集
- **削除**: 予定を長押ししてメニューから削除
- **コピー**: 同一タスクの予定をコピー（繰り返し会議用）
- **タスクへのアクセス**: 予定をクリックして関連タスクの編集モーダルを開く

### フィルター機能
- **タスク別フィルター**: 特定タスクの予定のみ表示（未完了タスクのみ表示）
- **日付範囲フィルター**: 開始日・終了日を指定してフィルター
- **過去/未来フィルター**: 「過去を表示」チェックボックスで切り替え
- **検索**: 予定タイトル、タスク名、場所、メモで検索（ハイライト表示）

### エクセルコピー機能
1. 日付のチェックボックスでコピーする日付を選択
2. 一括選択ボタンで全選択/全解除
3. エクセルコピーボタンから形式を選択
   - **表形式（複数列）**: タブ区切りで複数列に展開
   - **1セル形式（列挙）**: 1セルに列挙形式で貼り付け

### 予定の表示内容
- **日時**: 開始時刻、終了時刻（終了時刻がない場合は開始時刻のみ）
- **場所**: 場所が設定されている場合は表示
- **タイトル**: 予定のタイトル
- **タスク名**: 関連するタスク名（クリック可能）
- **メモ**: メモがある場合は折りたたみ表示（デフォルトで閉じる）
- **過去の予定**: 過去の予定はグレーアウト表示

### リマインダー機能
- 予定時刻15分前にポップアップモーダル表示（Outlook風）
- 通知音付きのデスクトップ通知

### 「今日」へのジャンプ
- 予定表を開いた際、自動的に今日の位置にスクロール

## Gmail連携機能

### 機能概要
- タスク編集モーダルからGmailメーラーを起動
- APIやアクセストークンは不要
- メール送信履歴の記録

### 使用方法
1. タスク編集モーダルを開く
2. 「Gmailで送信」ボタンをクリック
3. Gmailメーラーが起動
4. メール送信後、送信履歴が記録される

### 設定
- 設定画面の「Gmail連携」セクションで説明を確認
- 接続テスト機能あり

## ヘルプセンター
- リンク管理画面・タスク管理画面の3点メニューから起動
- 見出し一覧（目次）とキーワード検索で目的の項目へすぐ移動可能
- 「HTML出力・印刷」ボタンで内容をブラウザに表示し、そのまま印刷やPDF保存が可能
- マニュアル中の「スクリーンショットを見る」リンクをクリックすると、`assets/help/` フォルダに配置した画像をポップアップ表示できます（画像ファイルは用途に合わせて各自で配置してください）
- 「動画を見る」リンクをクリックすると、`assets/help/videos/` に配置した MP4 ファイルをアプリ内で再生できます（存在しない場合は警告が表示されます）。

### 通知設定

- 通知音の有効/無効を設定
- 通知音のテストが可能

## デモ動画の作り方
1. **録画する**  
   - Windows 11 標準の *Xbox Game Bar*（`Win + G`）で「キャプチャ」ウィジェット → 「録画開始」を押す  
   - もしくは Clipchamp / OBS Studio など任意の録画ソフトを使用
2. **編集する（任意）**  
   - 不要な部分をトリミングし、字幕やハイライトを追加すると見やすさが向上
3. **出力する**  
   - MP4（H.264 + AAC）形式、解像度 1920x1080 / 60fps 以下を推奨  
   - ファイルサイズが 50MB 程度以下だと読み込みが安定
4. **配置する**  
   - `assets/help/videos/` フォルダに以下の名前で保存  
     - `demo_link.mp4` … リンク管理画面デモ  
     - `demo_task.mp4` … タスク管理画面デモ  
     - `demo_schedule.mp4` … スケジュール一覧デモ  
   - 別名を利用したい場合は `lib/views/help_center_screen.dart` の `_videoMap` を編集
5. **ビルドする**

## トラブルシューティング

### よくある問題と解決方法

#### Q: リンクが開かない
**A:**
- ファイル/フォルダのパスが正しいか確認
- ファイル/フォルダが存在するか確認
- URLの形式が正しいか確認

#### Q: ドラッグ＆ドロップが動作しない
**A:**
- 管理者権限でアプリを実行
- アンチウイルスソフトの設定を確認
- ファイル/フォルダのアクセス権限を確認

#### Q: データが消えた
**A:**
- エクスポートしたバックアップファイルから復元
- アプリの再起動を試行
- 設定ファイルの確認

#### Q: アプリが起動しない
**A:**
- 管理者権限で実行
- 必要なランタイムがインストールされているか確認
- アンチウイルスソフトの除外設定

#### Q: 通知音が鳴らない
**A:**
- Windowsのシステムサウンド設定を確認
- システムの音量設定を確認
- 設定画面で通知音が有効になっているか確認
- 通知音テストボタンで動作確認
